# 🚀 Zerodha Clone

A full-stack stock trading platform inspired by Zerodha, built using the MERN stack. This project recreates the core structure and user experience of the Zerodha platform, including a responsive landing page, user dashboard, and backend API.

> **Note:** This project is built for educational purposes only and is not affiliated with or endorsed by Zerodha.

---

## 📸 Project Overview

This repository contains three main applications:

* **Frontend** – Responsive landing website
* **Dashboard** – User dashboard for managing investments and portfolio
* **Backend** – REST API built with Node.js, Express, and MongoDB

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/mahedihshayek/ZerodhaClone.git
cd ZerodhaClone
```

### 2. Install dependencies

For each application:

```bash
cd frontend
npm install
```

```bash
cd dashboard
npm install
```

```bash
cd backend
npm install
```

---

## ▶️ Running the Project

### Start Backend

```bash
cd backend
npm start
```

### Start Frontend

```bash
cd frontend
npm start
```

### Start Dashboard

```bash
cd dashboard
npm start
```
